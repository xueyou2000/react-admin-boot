# react-admin-boot

一个类似`creat-react-app`的脚手架, 用于快速搭建一个`React` + `TypeScript`项目，开箱即用。

特点:

-   统一项目结构
-   根据环境变量启用不同配置

## 项目结构

-   src
    -   main.tsx
-   resources
    -   application.yml
    -   application-dev.yml
    -   template.html
-   mocks
-   static
-   config
    -   postcss.config.js
    -   webpack.config.js
-   dist
